  • +=(_:_:)

    Adds two NSAttributedStrings and stores the result in the left-hand-side.

    var hello = NSAttributedString(string: "Hello")
let world = NSAttributedString(string: " world!")
hello += world
print(hello.string)
// Prints "Hello world!"
