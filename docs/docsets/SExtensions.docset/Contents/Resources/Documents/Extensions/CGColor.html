CGColor

class CGColor : _CFObject
  • isEqualWithConversion(to:)

    Returns true if self is equal to another CGColor with different color space.

    let clearColor = UIColor(red: 0, green: 0, blue: 0, alpha: 0).cgColor
print(clearColor.isEqualWithConversion(to: UIColor.clear.cgColor))
// Prints "true"

    Declaration

    Swift

    func isEqualWithConversion(to other: CGColor) -> Bool

    Parameters

    other

    Another CGColor.

    Return Value

    Returns true if self is the same color as another CGColor with different color space; otherwise returns false.

  • isKind(of:maximumFractionDigits:)

    Returns true if self has the same component values as another CGColor with a given maximum fraction digits for the comparison.

    let white = UIColor(red: 0.994, green: 0.994, blue: 0.994, alpha: 1).cgColor
print(white.isKind(of: UIColor.white.cgColor, maximumFractionDigits: 1))
// Prints "true"

    Parameters

    aColor

    Another CGColor.
    maximumFractionDigits

    Return Value

    Returns true when the component values match, otherwise returns false.
