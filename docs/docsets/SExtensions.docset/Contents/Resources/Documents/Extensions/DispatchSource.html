    Parameters

    delay

    The time from now at which the first timer event will be delivered, subject to the leeway and other considerations described above. The deadline is based on Mach absolute
    interval

    The repeat interval for the timer, or .never if the timer should fire only once.
    isRepeating

    Whether the handler will execute repeatly.
    queue

    A dispatch queue the handler will execute on.
    handler

    A handler to execute.
